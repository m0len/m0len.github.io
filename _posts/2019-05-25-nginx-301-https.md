## 问题

其中一台服务器使用 Nginx 作为网页服务器，由于配置了 301 跳转 https 导致一直无法通过验证，无法自动更新证书，最后证书过期好久了才发现。一番查找后得出一个解决方法。

## 解决方法

在 Nginx 各虚拟主机配置文件中的 http 块中加入以下代码：

``` nginx
location /.well-known/ {
    add_header Content-Type 'text/plain';
    root /网站目录完整路径;
}
```
